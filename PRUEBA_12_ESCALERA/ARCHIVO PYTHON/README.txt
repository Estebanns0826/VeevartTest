=====================================================================================================================
                                            JUEGO DE LA SERPIENTE - COMPLETO
=====================================================================================================================
Debes ejecutar el codigo en cualquier compilador, el codigo te pregunta si deseas tirar el dado, por lo que debes
poner "si" para que el programa tire el dado, cuando la posición llega a 25, se termina el juego y has terminado


La operación es simple, al ejecutar el archivo ESCALERAS_SERPIENTES.py se abrira el codigo escrito en lenguaje python
seguidamente se debe ejecutar el codigo en cualquier compilador, se preguntara al usuario si desea tirar el dado por lo 
que el debe escribir la palabra "si", seguidamente tira el dado, el programa por consola muestra cuanto saco tirando
el dado, seguidamente verifica si esta en una escalera para subir de posición del juego o en una serpiente para bajar de
posición. Cuando el usuario llega a la posición 25 se termina el juego
